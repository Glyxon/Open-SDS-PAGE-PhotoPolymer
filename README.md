# Open-SDS-PAGE-PhotoPolymer
Frugal photopolymerization protocol for SDS-PAGE gels. Replace TEMED/APS with Riboflavin + EDTA using low-cost RGB LED matrices (8x8). Achieve controlled polymerization in 15-20 min. Designed for independent labs, biohackers, and resource-limited settings. Robust science, DIY hardware, technical sovereignty.
